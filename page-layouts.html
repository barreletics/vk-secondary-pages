Three layers. Eight colors. Snap-together pages.

Every VK page is a stack: one Header, any number of Sections, one Footer. Each Section has a category color so a junior can scan at a glance: green = trust, blue = card grid, plum = editorial, slate = process, red = CTA, teal = form.

Want to build an About page? Header (amber Hero 1) + Editorial (plum) + Stat strip (plum) + CTA (red) + Footer (black). That's it. Same colors used on every section card below and inside every Page Type recipe.

Layer 01 · Headers
Header blocks — pick one hero per page.
11 distinct heroes across 6 families. Light heroes are the default for storytelling, service, and policy pages. Dark heroes are reserved for category PLPs and brand spotlights.
Hero 1 — Big Photo (full-bleed)
Single full-width image, ~520-640px tall. Used on storytelling pages with a hero-quality photo.
Header1a · Light
Big Photo · Copy below
Use for: About, Heritage, Studio Installations — when copy needs full readability under the photo.
View live demo → about-v2.html
Header1b · Light + dark scrim
Big Photo · Copy overlaid
Use for: Make Your Mark, brand campaigns, immersive storytelling. Copy sits on a darkened scrim over the image.
View live demo → make-your-mark.html
Hero 2 — Light 50/50 (Trade pattern)
Full-bleed 50/50 split. Left panel: #EDE8E2 with eyebrow + H1 + body + CTAs. Right panel: #fff with one of 5 variants.
Header2a · Stats Grid (4 boxes)
Light 50/50 · Stats Grid
Use for: Trade Program, Studio Pros, Audio Consultants, Careers — credibility / authority pages. 2x2 grid of big numbers + labels.
View live demo → trade-program.html
Header2b · Full Image
Light 50/50 · Full Image
Use for: Studio Professionals, service pages with one hero photo. Image fills the entire right panel — no padding.
View live demo → studio-professionals.html
Header2c · Framed Image
Light 50/50 · Framed Image
Use for: Playback, programs where the photo needs breathing room (white margin around it). Image contained, not full-bleed.
View live demo → playback.html
Header2d · Feature List
Light 50/50 · Feature List
Use for: Audio Consultants, programs that lead with a benefits list. Bullet list with amber dots, no images.

Header2e · Product Grid (Six-Card)
Light 50/50 · Product Grid

Hero 3 — HOF Pattern (gear detail)
Same 50/50 frame as Hero 2 but right panel has horizontal padding so the product photo ends flush at the bottom. Used for Hall of Fame gear pages.

Header4 · Light Centered
Text Only · Centered

Layer 01 · Headers — dark family
Dark headers — category PLPs and brand spotlights only.

Layer 02 · Sections
Section blocks — stack any number between header and footer.
Color-coded by category: green = trust, blue = card grid, plum = editorial, slate = process, red = CTA, teal = form. Each block has a wireframe, a "when to use" rule, and a live demo link.
