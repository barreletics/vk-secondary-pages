Why we do it this way

Manageable for the agency. Premium for the customer.

A consistent body rhythm with a varied hero is what makes CB2, Apple, and Sweetwater feel premium. We picked the same approach for VK: one shared body template across every secondary page, three hero variants chosen by page type, and two universal strips (trust above the hero, explore above the footer). Less drift. Higher conversion. Easier to maintain.

Manage
One body, every page
When something needs to change site-wide, you change it in the build script and re-run. No per-page custom CSS to track down.
Convert
Three heros, by intent
Hero A (Split Bleed) for HOF gear. Hero B (Split Contained) for programs and service. Hero C (Editorial) for brand-led storytelling. Body modules add wow shots on any hero.
Reinforce
Two strips, always
Trust strip below the hero builds confidence. Explore strip above the footer recovers exits and surfaces related pages.
One body

The shared rhythm — every page.

Section order is locked. Color shifts at every section boundary keep the eye moving and the page feeling fast.

  1. 1. Hero — #EDE8E2 warm light. Dark text, red eyebrow, red CTA button. Hero variant determined by page type.
  2. 2. Trust strip — #FFFFFF. 4 inline credentials, thin top + bottom border, no padding inside the items. Content preset by page family.
  3. 3. Grid / cards — var(--off-white) #F7F5F2 background. White cards with 1px rgba border. 3-up on desktop, 1-up on mobile.
  4. 4. Content / body — #FFFFFF background. Dark text. Long-form copy, deep links, JSON-LD blocks live here.
  5. 5. CTA / form — #EDE8E2 warm light, white inputs, red button. Only one CTA section per page.
  6. 6. Explore strip — var(--off-white) #F7F5F2. 3 cross-link cards. Content preset by page family.
  7. 7. Footer — Standard VK footer template (cloned via JS from <template id="footer-template">).

Decision rules

Pick a hero in 5 seconds.

Page typeGoalHeroBody modulesWhy
HOF gearBrowse → desire → buyHero ABig + spec strip + DualFull-bleed photo earns attention. Big body shot + dual shots build desire across the scroll.
Program / ServiceDecide → contactHero BDual (optional)One clear pitch on the left, one strong photo on the right, one CTA below.
EditorialRead → shareHero CBig or DualStory leads. The wow shot lives in the body via a module, not in the hero.
CampaignClick → shopHero Bnone (or Big)Promo balance — copy + image + CTA above the fold; optional body shot for hero product.
PolicyRead → complyHero BnoneLightweight hero, body does the heavy lifting. No dark heroes on policy pages.
Things you should never do
  • No dark heroes on secondary pages. All heros open with #EDE8E2.
  • No more than one dark accent block per page (#2C2C2A) — and never first.
  • No amber on new gear or program pages. Amber = vintage/used only.
  • No Playfair on prices. DM Sans 500 weight, --near-black only.
  • No pinkish reds. Banned: #fde8e6, rgba(192,57,43,0.04–0.08) as standalone backgrounds.
  • No font sizes below 12px anywhere — CSS, inline, or SVG labels.
  • No physical locations. VK has no showrooms — never reference a city as a VK location.
  • No new strips. Use one of the 5 presets above; new strip copy needs agency approval.
